## 🚀 使用方法

### 開発環境での起動

```bash
npm install    # 初回のみ
npm run dev    # 開発サーバー起動
```

ブラウザで `http://localhost:3000` にアクセス

### 操作フロー

#### 一般ユーザー
1. **病院コード** と **ID** と **PIN** を入力してログイン
2. カレンダービューで月を選択
3. 日付をクリックして「希望休」または「有給」を申請
4. **同じ病院内** の他のユーザーの申請は「〇名希望あり」として表示

#### 管理者
1. 病院コード（任意）、ID「管理者」、PIN「0000」でログイン
2. 「設定」ボタンでその病院の有給マスタを編集
3. 「管理者ビュー」ボタンでスタッフ全員の希望をマトリックス表示

#### 複数病院を管理する場合
1. 各病院コードでログイン（ブラウザが5つまで同時セッション可）
2. 病院ごとに独立したデータを確認・管理
